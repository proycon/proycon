## Some notable projects

* [STAM](https://annotation.github.io/stam/) - A data model and tooling for stand-off annotations on text
* [Textsurf](https://codeberg.org/proycon/textsurf) - Webservice for efficiently serving multiple plain text documents or excerpts thereof (by unicode character offset), without everything into memory.
* [Analiticcl](https://codeberg.org/proycon/analiticcl) - An approximate string matching or fuzzy-matching system for spelling correction, normalisation or post-OCR correction
* [Colibri-Core](https://codeberg.org/proycon/colibri-core) - An NLP tool as well as a C++ and Python library for working with basic linguistic constructions such as n-grams and skipgrams (i.e patterns with one or more gaps, either of fixed or dynamic size) in a quick and memory-efficient way.
* [Codemetapy](https://codeberg.org/proycon/codemetapy) - A Python package for generating and working with codemeta.
* [SXMO](https://sxmo.org) - A minimalist desktop environment for Linux mobile devices
    * [wvkbd](https://git.sr.ht/~proycon/wvkbd) - a virtual keyboard for wayland
* [Vocage](https://git.sr.ht/~proycon/vocage) - A minimalistic spaced-repetion vocabulary trainer (flashcards) for the terminal
* [Todo.txt more](https://git.sr.ht/~proycon/todotxt-more) - Extensions for todo.txt: interactive rofi/fzf control, sync github issues, better colors, time tracking... and more!
* [My CLI tools for text processing](https://proycon.anaproy.nl/posts/my-cli-tools-for-text-processing/) - A blog posts where I link to various NLP tools I wrote over the years

You may also want to look at my [dotfiles](https://codeberg.org/proycon/dotfiles)
